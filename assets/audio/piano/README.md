# Piano Samples for Chord Generator

This folder should contain the following piano sample files:

- C3.mp3
- E3.mp3
- G3.mp3
- C4.mp3
- E4.mp3
- G4.mp3
- C5.mp3

## Free Piano Sample Sources

You can download free piano samples from these sources:

1. **Freesound.org**
   - Search for "piano C3", "piano E3", etc.
   - Filter by Creative Commons licenses

2. **Philharmonia Orchestra Samples**
   - https://philharmonia.co.uk/resources/sound-samples/
   - Free piano samples in various notes

3. **University of Iowa Electronic Music Studios**
   - http://theremin.music.uiowa.edu/MISpiano.html
   - Professional piano recordings

## How to Use

1. Download the 7 piano note samples (C3, E3, G3, C4, E4, G4, C5)
2. Rename them to match the required filenames above
3. Place them in this folder (assets/audio/piano/)
4. The chord generator will automatically load and use them for playback

The system uses pitch-shifting to play all notes from these 7 samples, providing realistic piano sound for the entire chord progression.
